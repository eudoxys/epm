# Eudoxys package manager

## Installation

To install the Eudoxys package manager

    python3 -m venv .venv
    . .venv/bin/activate
    pip install git+https://github.com/eudoxys/epm

## Usage

To get the index of available packages

    epm index

To get a list of installed packages

    epm list

To install a package

    epm install NAME
